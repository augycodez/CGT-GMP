About

Built for GMP Professionals

The Global GMP Intelligence Platform is an open, community-maintained reference library for Good Manufacturing Practice guidelines. It is designed specifically for GMP practitioners in cell and gene therapy, biologics, and pharmaceutical manufacturing — with a focus on ease of access, comparative navigation, and offline-ready deployment.

Built in plain HTML, CSS and JavaScript — no frameworks, no database. Runs locally in VS Code Live Server or from any static web host. Add your own PDFs to the pdfs/ folders and reference them in data/guidelines.json.
